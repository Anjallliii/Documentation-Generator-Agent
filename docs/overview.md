# Project Documentation – Documentation Generator Agent

## Overview
Documentation Generator Agent is an AI-powered system that automatically generates structured and readable documentation from source code repositories.

## How It Works
1. Takes source code as input
2. Analyzes files and structure
3. Uses a Large Language Model to understand logic
4. Generates documentation in Markdown format

## Architecture
Input Code → Code Parser → LLM → Markdown Documentation

## Target Users
- Software Developers
- Students
- Open-source contributors
