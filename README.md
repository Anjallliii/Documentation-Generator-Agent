# Documentation Generator Agent

An AI-powered Documentation Generator Agent that automatically generates structured and developer-friendly documentation from source code.

## Features
- Automatic README generation
- Code explanation using LLMs
- Markdown-based documentation
- Easy to use and extend

## Tech Stack
- Python
- LLM (OpenAI/Gemini)
- LangChain
- Markdown

## Use Case
Helps developers and open-source contributors save time by automating documentation.
